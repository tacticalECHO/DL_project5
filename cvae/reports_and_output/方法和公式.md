# CVAE Inpainting 方法技术文档

> **目的**：供报告撰写人参考。内容包含问题形式化、最终方法的公式推导、架构细节、训练配置、评估协议与结果。
> **对应代码版本**：v1.3（条件先验 + 无 compose）

---

## 1. 任务形式化

### 1.1 问题定义

令 $x \sim p_X$ 为一张自然人脸图像，$M \in \{0, 1\}^{H \times W}$ 为二值矩形掩码（$M_{ij}=1$ 表示该像素被遮挡），则受损图像为：

$$
y = x \odot (1 - M)
$$

其中 $\odot$ 表示逐元素相乘。我们的任务是学习条件分布 $p_{\hat{X} \mid Y, M}$，并从中采样得到修复图 $\hat{x}$。

### 1.2 评估尺度

测试时掩码中心 $(c_x, c_y)$ 在图像内均匀采样，尺寸 $(h, w)$ 在三档区间内均匀采样：


| Regime | Height / Width range               | 覆盖比例 |
| ------ | ---------------------------------- | -------- |
| Small  | $[0.1H, 0.2H] \times [0.1W, 0.2W]$ | ~1–4%   |
| Medium | $[0.3H, 0.4H] \times [0.3W, 0.4W]$ | ~9–16%  |
| Large  | $[0.5H, 0.6H] \times [0.5W, 0.6W]$ | ~25–36% |

三档掩码对每张 test 图固定保存（`data/fixed_masks_{regime}.pt`），保证组内所有方法的评估可比。

### 1.3 两类指标（Blau & Michaeli CVPR 2018）

Blau & Michaeli 证明：任何修复算法无法同时最小化 **distortion** $\mathbb{E}[\Delta(X, \hat{X})]$ 和 **perception** $d(p_X, p_{\hat{X}})$。本项目的四个指标按此分类：

- **Distortion**: MSE, PSNR, SSIM （配对像素/结构距离）
- **Perception**: FID, LPIPS（分布距离与 feature-space 距离）

## 2. CVAE 方法（v1.3 最终方案）

### 2.1 Conditional ELBO 推导

条件 VAE 将隐变量 $z$ 引入生成过程。令条件 $c = (y, M)$，则期望的对数似然为：

$$
\log p(x \mid c) = \log \int p_\theta(x \mid z, c) \, p(z \mid c) \, dz
$$

直接优化该似然困难（积分不可解），引入变分分布 $q_\phi(z \mid x, c)$，可得 ELBO（Evidence Lower BOund）：

$$
\log p(x \mid c) \;\geq\; \mathbb{E}_{q_\phi(z \mid x, c)}\big[\log p_\theta(x \mid z, c)\big] - D_{KL}\big(q_\phi(z \mid x, c) \,\|\, p_\psi(z \mid c)\big)
$$

等号成立当且仅当 $q_\phi(z \mid x, c) = p(z \mid x, c)$。我们最大化 ELBO，等价于最小化负 ELBO：

$$
\mathcal{L}_{\text{ELBO}}(x, c) = -\mathbb{E}_{q_\phi(z \mid x, c)}\big[\log p_\theta(x \mid z, c)\big] + D_{KL}\big(q_\phi(z \mid x, c) \,\|\, p_\psi(z \mid c)\big)
$$

### 2.2 具体损失形式

**(1) 重建项**：我们使用 **L1 loss** 代替原始高斯似然的 log 形式：

- 原始 NLL 若假设 $p_\theta(x \mid z, c) = \mathcal{N}(x; \hat{x}, \sigma^2 I)$，则 $-\log p_\theta \propto \|x - \hat{x}\|_2^2$
- 改用 L1 是因为 L2 会产生过平均化的模糊输出，L1 在实践中更锐利
- 对应假设 $p_\theta$ 为拉普拉斯分布：$p_\theta(x \mid z, c) = \text{Laplace}(x; \hat{x}, b)$，负对数似然 $\propto \|x - \hat{x}\|_1$

$$
\mathcal{L}_{\text{rec}}(x, c) = \|\hat{x} - x\|_1, \quad \hat{x} = \text{Decoder}(z, y, M)
$$

**(2) KL 项**：$q_\phi$ 和 $p_\psi$ 均为对角高斯，有闭式解：

$$
q_\phi(z \mid x, c) = \mathcal{N}\big(z; \mu_q(x, c), \text{diag}(\sigma_q^2(x, c))\big)
$$

$$
p_\psi(z \mid c) = \mathcal{N}\big(z; \mu_p(c), \text{diag}(\sigma_p^2(c))\big)
$$

两个对角高斯的 KL 闭式为（逐维度求和）：

$$
D_{KL}(q_\phi \| p_\psi) = \sum_{i=1}^{D} \left[ \log \frac{\sigma_{p,i}}{\sigma_{q,i}} + \frac{\sigma_{q,i}^2 + (\mu_{q,i} - \mu_{p,i})^2}{2 \sigma_{p,i}^2} - \frac{1}{2} \right]
$$

**(3) 最终损失（带 $\beta$-warmup）**：

$$
\boxed{\;\mathcal{L}(x, c) = \|\hat{x} - x\|_1 + \beta(t) \cdot D_{KL}\big(q_\phi(z \mid x, c) \,\|\, p_\psi(z \mid c)\big)\;}
$$

其中 $\beta(t)$ 为 KL 权重，采用线性 warm-up：

$$
\beta(t) = \min\left(\beta_{\max},\ \beta_{\max} \cdot \frac{t}{T_{\text{warmup}}}\right)
$$

$\beta_{\max} = 1.0$，$T_{\text{warmup}} = 10$ epoch。Warm-up 的目的是避免训练初期 KL 项主导导致 posterior collapse。

### 2.3 网络架构

三个独立的 `nn.Module`：

#### Encoder $q_\phi(z \mid x, c)$（23.74M 参数）

```
输入: concat(gt, masked, mask) → [B, 7, 256, 256]
  Conv(7→64,    k4 s2) + GroupNorm(8) + SiLU  → [B,  64, 128, 128]
  Conv(64→128,  k4 s2) + GroupNorm(8) + SiLU  → [B, 128,  64,  64]
  Conv(128→256, k4 s2) + GroupNorm(8) + SiLU  → [B, 256,  32,  32]
  Conv(256→512, k4 s2) + GroupNorm(8) + SiLU  → [B, 512,  16,  16]
  Conv(512→512, k4 s2) + GroupNorm(8) + SiLU  → [B, 512,   8,   8]
  Flatten → Linear(32768 → 512)
  → μ_q ∈ ℝ^256,  logσ²_q ∈ ℝ^256
```

#### PriorNet $p_\psi(z \mid c)$（9.95M 参数）

```
输入: concat(masked, mask) → [B, 4, 256, 256]
  Conv(4→32,    k4 s2) + GroupNorm(8) + SiLU  → [B,  32, 128, 128]
  Conv(32→64,   k4 s2) + GroupNorm(8) + SiLU  → [B,  64,  64,  64]
  Conv(64→128,  k4 s2) + GroupNorm(8) + SiLU  → [B, 128,  32,  32]
  Conv(128→256, k4 s2) + GroupNorm(8) + SiLU  → [B, 256,  16,  16]
  Conv(256→256, k4 s2) + GroupNorm(8) + SiLU  → [B, 256,   8,   8]
  Flatten → Linear(16384 → 512)
  → μ_p ∈ ℝ^256,  logσ²_p ∈ ℝ^256
  (对 logσ²_p 施加软下限 clamp(min=-2.0)，防止 prior 方差退化)
```

#### Decoder $p_\theta(x \mid z, c)$（15.71M 参数）

```
输入: z ∈ ℝ^256
  Linear(256 → 32768) → Reshape → [B, 512, 8, 8]

  5 个 Upsample Block，每个 block：
    NearestUpsample(×2) → concat(下采样到当前分辨率的 (masked, mask))
                       → Conv(k3 s1) + GroupNorm(8) + SiLU
                       → Conv(k3 s1) + GroupNorm(8) + SiLU

  Block 1: [B, 512, 8,  8]  → [B, 512, 16,  16]
  Block 2: [B, 512, 16, 16] → [B, 256, 32,  32]
  Block 3: [B, 256, 32, 32] → [B, 128, 64,  64]
  Block 4: [B, 128, 64, 64] → [B,  64, 128, 128]
  Block 5: [B,  64, 128,128]→ [B,  32, 256, 256]

  Conv(32 → 3, k3 s1) + Sigmoid → x̂ ∈ [0, 1]^{3×256×256}
```

**关键架构选择**：

- **无 U-Net skip connection**：避免 encoder 特征绕过 latent 瓶颈，保证 $z$ 承载信息
- **多尺度条件注入**：$(y, M)$ 在每个上采样 scale 下采样后 concat 进 decoder，提供边界参考
- **Sigmoid 输出**：数据值域 $[0,1]$，匹配数据集 `ToTensor()` 的输出
- **不做 compose**：decoder 直接输出整张图 $\hat{x}$，而非只生成 hole 区域。原因见 §3.1 设计迭代记录。

**总参数量**：49.40M

### 2.4 训练与推理

**训练阶段**：$z$ 从 posterior 采样（reparameterization trick）

$$
z = \mu_q + \sigma_q \odot \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)
$$

**推理阶段**：$z$ 从条件先验 $p_\psi$ 采样（不是标准正态）

$$
z \sim \mathcal{N}(\mu_p(y, M), \text{diag}(\sigma_p^2(y, M)))
$$

**Pluralistic 生成**：对同一输入 $(y, M)$ 重复采样多次 $z$，可得到多个不同的修复结果，展示 CVAE 的随机性。

## 3. 设计迭代记录（可选写入 Report Discussion）

本节记录设计演化过程。如 report 篇幅受限，可浓缩为一段话；篇幅允许则作为独立 subsection 增强说服力。

### 3.1 v1.2（失败尝试）：标准先验 + Compose

初始设计采用最简化 CVAE 形式：

- 固定先验 $p(z) = \mathcal{N}(0, I)$
- Compose 输出：$\hat{x} = y \odot (1 - M) + \hat{x}_{\text{raw}} \odot M$（只生成 hole，已知区完美保留）
- KL 为标准闭式 $D_{KL}(q_\phi(z \mid x, c) \| \mathcal{N}(0, I))$

**失败现象**：训练 5 epoch 内 KL 从 1500 断崖式下降至 $<1$，且 validation loss 不随 epoch 改善。

**诊断**：Posterior collapse。原因是 decoder 通过多尺度空间条件 $(y, M)$ 可直接重建 hole 边界附近的像素，不依赖 $z$。而优化器为降低 KL，将 $q_\phi$ 硬拉向标准先验，使 $z$ 退化为纯噪声。

**补救尝试（Free Bits）**：对每维 KL 设下限 $\lambda = 0.5$ nats（Kingma et al. 2016 IAF）。结果：posterior 容量被保住（KL 稳定在 $D \cdot \lambda = 128$ nats），但 val_l1 仍不改善——说明 decoder 端已学会**无视 $z$**，只用空间条件做确定性填充。Encoder 端的修复对此无效。

### 3.2 v1.3（当前方案）：条件先验 + 无 Compose

针对 v1.2 的根本问题，做了两处修改：

**(a) 去除 compose**：decoder 必须重建整张图（包括已知区），$z$ 对所有像素都负责。这从根本上消除了 "decoder 可以忽略 $z$" 的可能性。

**(b) 引入条件先验 $p_\psi(z \mid y, M)$**：

- $q_\phi$ 和 $p_\psi$ 都是可学习的，KL 项成为"两个网络输出之间的差异"，而非"把 $q_\phi$ 拉向固定分布"
- $p_\psi$ 看到条件 $c$，可以将每个 $(y, M)$ 映射到合适的 latent 分布区域
- 参考 Pluralistic Image Completion（Zheng et al. CVPR 2019）的 conditional prior 设计，本实现为其 GAN-free 简化版

**验证**：v1.3 第一次 smoke training 即观察到 val_l1 稳定下降，训练 50 epoch 后 best val L1 = 0.0132，medium 档 FID = 7.93（见 §5 结果）。

### 3.3 总结

这一迭代不仅说明了"标准 CVAE 在 inpainting 上为何无效"，也揭示了 Pluralistic Completion 论文中 conditional prior 设计的**必要性**——它不是一个可有可无的架构选择，而是防止 posterior collapse 的关键。

## 5. 评估协议与结果

### 5.1 指标定义

令 $N$ 为 test set 大小。


| 指标  | 定义                                                                                      | 类型            |
| ----- | ----------------------------------------------------------------------------------------- | --------------- |
| MSE   | $\frac{1}{N} \sum_i \| x_i - \hat{x}_i \|_2^2 / (H \cdot W \cdot C)$                      | Distortion      |
| PSNR  | $10 \log_{10}(1 / \text{MSE}_i)$ 后平均                                                   | Distortion      |
| SSIM  | Structural Similarity Index, data_range=1.0                                               | Distortion      |
| LPIPS | $\frac{1}{N} \sum_i d_{\text{AlexNet}}(x_i, \hat{x}_i)$, 输入先归一化到 $[-1,1]$          | Perception-side |
| FID   | Fréchet distance between Inception feature distributions of$\{x_i\}$ and $\{\hat{x}_i\}$ | Perception      |

**为什么选 LPIPS 作为第四指标**：

- LPIPS (Zhang et al. CVPR 2018) 在 learned perceptual feature space 测距离，与人类判断相关性高
- 与 FID 互补：FID 是分布级距离，LPIPS 是样本级配对距离
- 严格按 Blau-Michaeli 定义，perception metric 应为无参考的 $d(p_X, p_{\hat{X}})$；LPIPS 需要 ground truth 配对，本质为 feature-space distortion。但 learned features 使其比 pixel-space 指标更接近 perception 侧，社区普遍如此使用。本 report 中将其报告在 perception-side，并在分析中注明这一边界性质。

### 5.2 CVAE 测试集结果

以 `checkpoints/run_01/best.pt`（epoch 46，val L1 = 0.01323）在 test set（3000 张 × 3 regime）评估：


| Regime | MSE      | PSNR  | SSIM   | LPIPS  | FID   |
| ------ | -------- | ----- | ------ | ------ | ----- |
| Small  | 0.000253 | 38.00 | 0.9857 | 0.0116 | 1.57  |
| Medium | 0.002073 | 27.85 | 0.9414 | 0.0660 | 7.93  |
| Large  | 0.006575 | 22.57 | 0.8563 | 0.1669 | 25.99 |

**观察**：

- 所有指标随 mask 尺寸单调变化：distortion 指标（MSE、1-SSIM、1-PSNR）随 hole 增大而增大；perception 指标（LPIPS、FID）同步增大。符合直觉（越大的 hole 越难重建且越难匹配分布）。
- Medium 档 FID = 7.93 属于较强水平（Pluralistic Completion 原论文在 CelebA-HQ 上 FID ≈ 10–15，我们的简化版未加对抗训练仍达到此水平）。
- 视觉检查可见关键面部区域（眼、嘴、眉毛）存在轻微模糊，属于 L1 重建损失的固有特性（L1 鼓励所有合理补全的像素均值）。

### 5.3 Perception-Distortion Tradeoff 分析框架

三点在 P-D 平面上从 small → medium → large 形成单调递增轨迹（distortion 和 perception 同向增大），这是 L1-reconstruction CVAE 的典型表现：在每个难度层次，模型优先保证 distortion 最优，perception 作为妥协产物。

当与 Reconstruction 和 GAN 方法对比时，CVAE 预期位置：

- **相对 Reconstruction**：distortion 相近或略差（因为随机性），perception 应显著更优（因为 $z$ 提供随机性，输出分布更接近真实分布）
- **相对 GAN**：distortion 更优（L1 保证像素误差），perception 略差（GAN 直接优化分布距离）

此位置符合 Blau-Michaeli 的理论预测：无对抗训练的 likelihood-based 方法位于 P-D 曲线的"低 distortion 端"，GAN 方法位于"低 perception 端"。

### 5.4 Pluralistic 采样验证

CVAE 相对 reconstruction 方法的核心优势是随机性。对同一 masked 输入采样 5 个不同 $z$，pairwise pixel 差异平均约 0.01（值域 $[0,1]$ 内），证明模型输出确有多样性（而非退化为确定性函数）。可视化见 `reports/pluralistic_grid_large.png`。
